## Description

Use the GTIN attribute to submit a [Global Trade Item Number (GTIN) | GS1](https://www.gs1.org/standards/id-keys/gtin). A GTIN uniquely identifies your product.

- GTIN enables automated product classification therefore any new products added to the feed will be indexed automatically
- We're strictly following [UPC](https://en.wikipedia.org/wiki/Universal_Product_Code#Composition)/[EAN](https://en.wikipedia.org/wiki/International_Article_Number#Composition)/[JAN](https://en.wikipedia.org/wiki/International_Article_Number#jan)/[ISBN](https://en.wikipedia.org/wiki/International_Standard_Book_Number) and [ITF-14](https://en.wikipedia.org/wiki/ITF-14)
  
:::tip Effects When Used

Including this will help indexing your products faster on Pricespy.

:::

:::caution Effects When Omitted

Process time will be slower.

:::

## Validation Rules

If any of these rules are broken, it is the same as if this field would be *omitted*. Please see [severity levels](../appendix/severity-levels) for more details.

1. Use only one attribute of this type per product
1. Length should be one of `0, 8, 12, 13, or 14` digits (UPC, EAN, JAN, or ISBN)
1. Make sure that the check digit is present and correct using the [GS1 Check digit calculator](https://www.gs1.org/check*digit*calculator)
1. Don’t submit a GTIN in the restricted ranges. Restricted ranges have the following prefixes: `02, 04, or 2`
1. Don’t submit a GTIN in the coupon ranges. Coupons have the following GS1 prefixes ranges: `05, 98–99`
1. Must be a integer number without decimals
